# MeetFlow Desktop Wrapper (Electron)

Este diretorio prepara a versao desktop para uso no notebook corporativo.

## Como funciona no MVP

- Carrega a aplicacao web local (`http://localhost:3000`) em janela nativa.
- Mantem experiencia desktop sem alterar o core do sistema.

## Proximo passo sugerido

- Empacotar com `electron-builder` ou migrar para `Tauri` para reduzir consumo.
